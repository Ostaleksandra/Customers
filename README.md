# Список гостей

Учебный проект на Python.

## Описание

Программа создаёт класс `Customers`, который хранит информацию о клиенте: имя, фамилию, город и баланс.
Из объектов класса формируется список гостей и выводится краткая информация о каждом из них.

## Использование

После запуска программа выводит список гостей в формате:

```
Имя Фамилия. г. Город
```

## Цель проекта

Практика работы с:

* классами в Python
* методами
* списками объектов
